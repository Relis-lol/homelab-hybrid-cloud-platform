# Homelab Hybrid Cloud Platform

This project documents the step-by-step evolution of my private Linux server into a hybrid cloud platform.

Current baseline:
- Ubuntu Server installed
- SSH access configured
- Firewall (UFW) enabled
- System updated & secured

Goal:
Transform this server into a production-style platform including:
- API services
- Database backend
- Public web dashboard (EVE Online market history)
- Monitoring & access logging
- Azure integration
- CI/CD pipeline
- Architecture documentation

---

## Roadmap

### Phase 1 – Linux Baseline (Current)
- OS installation
- SSH setup
- Firewall configuration

### Phase 2 – Core Services
- Database setup
- API layer
- Secure access model

### Phase 3 – Public Web Dashboard
- Market data integration (EVE Online)
- Graph-based visualization
- Public read-only interface

### Phase 4 – Observability
- Access logs
- Performance monitoring
- Resource tracking

### Phase 5 – Azure Hybrid Integration
- Azure resource integration
- Cloud storage / services
- Secure connection model

### Phase 6 – CI/CD & Automation
- GitHub Actions
- Deployment workflows
- Infrastructure validation
